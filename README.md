# Beacon
As a submission Assistive Innovation Challenge 2026, beacon is a project wholeheartedly aimed to propagate and amplify the support system of someone suffering through Anorexia Nervosa.

## Features
| Feature | Description |
| ------- | ----------- |
| **Conversations with a support system** | A therapist, dietitian, family, and friends can see when a patient is doing well, needs company, or to simply check on then. |
